#app.py 
    *the head of the flask application, nothing else other than initilize flask object.

#config.py
    *all config parameters are stored config.py in the main directory.

#api.py
    *handles all the checking and redirections.

#errors.py
    *all error checking functions must be written in this file.

#server.py
    *handles all the processing, can only be invoked by api

#run.py
    *run this file to start the program.
    *alter parametes in config.py

#lib
    *contains util functions and sql file.

#tests
    *stores all the unit test files.

#features
    *stores all extra python scripts/programs.
